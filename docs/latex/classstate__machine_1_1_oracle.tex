This class should simulate what happens when the robot has collected all the hints that forms a complete and consistent hypothesis, and it tries its guess. This class should inform the robot if the hypothesis found is the winning one or not, and this is done thanks to the oracle\+\_\+service that checks the ID of the hypothesis of the robot with the winning one. If the hypothesis is correct then the game is finished. If it is not then the robot should restart searching for new hints and repeat all the process. Here we have two outcomes\+:
\begin{DoxyItemize}
\item game\+\_\+finished, if the hypothesis found is the correct one
\item motion, if the hypothesis found is wrong. ~\newline
