This class should simulate the movement of the robot between the various rooms of the cluedo game. If the hints percieved are less than the expected number given by the subscriber, then the robot should keep going searching hints in other rooms. If, instead, the number of hints collected is the right one the the class should check if the hypothesis formed with those hints is complete and consistent. If it\textquotesingle{}s not the robot should restart searching for collecting other hints, instead, if the hypothesis is complete and consistent the robot can go to the oracle room trying its guessing. There are three outcomes\+:
\begin{DoxyItemize}
\item enter room, if the robot is still collecting hints
\item go\+\_\+oracle, if the hypothesis formulate is complete and consistent
\item moving, if the hypothesis formulated is uncomplete or inconsistent. ~\newline
 
\end{DoxyItemize}
